# Enterprise-Style-Angular-Portfolio
**professional, enterprise-grade Angular portfolio website** for you! This will feature clean minimal layouts, premium business styling, and modern animations. Perfect for showcasing your skills at https://github.com/Dev-moe-kyawaung/

# 🚀 Build & Deploy Commands

```bash
# Development
ng serve --open

# Build for production
ng build --prod

# Deploy to GitHub Pages
npm install -g angular-cli-ghpages
ng build --prod --base-href="https://Dev-moe-kyawaung.github.io/portfolio-angular-enterprise/"
 ngh --dir=dist/portfolio-angular-enterprise
```
